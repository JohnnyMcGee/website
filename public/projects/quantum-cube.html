QUANTUM CUBE

Cloud-based Speed Cubing Timer App

Quantum Cube stores and protects its users' most precious data -- their speed-cubing solve times. It was the first app of its kind to store solve times in the cloud, making them available across all devices. Cubers can organize their solves into sessions, visualize their statistics, and prepare for competitions with the help of the Quantum Cube app.

Backend Node.js, Express, Redis, PostgreSQL
Web Frontend React, Redux, Sass
Mobile Frontend Flutter, Bloc, Isar DB
Puzzles Supported 2x2x2, 3x3x3, 4x4x4, 5x5x5, 6x6x6, 7x7x7, Pyraminx, Megaminx, Skewb, Square-1, Clock

Cloud Storage for Solve Times

Quantum Cube's source of truth is a cloud-hosted PostgreSQL database. A Redis server provides server-side session storage and some caching. Web and mobile clients sync solve times to the server, where they are stored securely.

React Single-Page Application

The timer UX is smooth and responsive, with a minimalist design that puts the focus on the solve. User sessions tend to be long, with solve data and statistics continuously updating in real-time. The SPA architecture allows for a seamless user experience on the web platform.

Flutter Mobile App

The Quantum Cube mobile apps allow cubers to take their solves on the go. Solve times are stored locally on the device, then automatically sync to the backend whenever an internet connection is available. Google's Flutter framework compiles to native code for iOS and Android, providing a consistent 60-120fps experience across all devices.
